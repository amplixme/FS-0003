# FS0003-3

Pasos que hice:

1. Cree la estructura de carpetas en `src`.
2. Agregue `controllers`, `middlewares`, `routes` y `services`.
3. Deje un `.gitkeep` en cada carpeta vacia.
4. Deje `app.js` para configurar Express.
5. Deje `server.js` para levantar el servidor.

Para probar:

1. Instalar dependencias con `npm install`.
2. Levantar el server con `npm run dev`.
3. Revisar que no tire error.
